Definition

- Media queries is a feature of CSS used to apply conditional properties based on a user’s device.
- Media queries allow developers to create responsive designs by adapting to different layouts, fonts, images, screen sizes, and orientations.
- A media query is a logical expression that is either true or false.
- A media query is true if the media type matches the type of the device where the user’s device and all expressions in the media query are true.
- I have used abit of media queries on this document to serve as a demonstration on the capabilities of Media queries.
